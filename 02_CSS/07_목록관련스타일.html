        [ 표현법 ]

        * 순서 없는 목록(ul)
        선택자{
            list-style : disc / circle / square / none;
        }

        * 순서 있는 목록(ol)
        선택자{
            list-style : decimal(십진수)
            list-style : lower-roman / upper-roman => 로마숫자
            list-style : lower-alpha / upper-alpha => 알파벳
        }
